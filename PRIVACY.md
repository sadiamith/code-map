# Privacy Policy

**code-map collects no data.**

code-map runs entirely on your machine. It scans your local repository and writes a
single self-contained HTML file. It does not:

- collect, store, or transmit any personal information;
- send your code, file paths, or generated maps anywhere;
- make any network requests — the generator uses only the local filesystem, and the
  produced HTML loads no external resources (no remote fonts, scripts, or analytics);
- include any telemetry, tracking, or third-party services.

Everything the plugin produces stays in the directory where you run it. Because no data
ever leaves your device, there is nothing for code-map to share, sell, or retain.

Questions: open an issue at https://github.com/sadiamith/code-map/issues
